# 心晴日记 — 技术架构规范

> 版本：v1.0 | 更新：2026-05-12

## 一、技术栈

### 前端：Flutter (Dart)
| 层 | 技术 |
|----|------|
| 状态管理 | Provider |
| KV存储 | shared_preferences |
| 结构化存储 | sqflite（照片路径、AI精灵数据） |
| 图表 | fl_chart |
| 网络 | http |
| 图片选择 | image_picker |
| 本地通知 | flutter_local_notifications |
| 音频 | audioplayers |
| Canvas | CustomPainter（天气动画、精灵、图标） |
| 触感 | HapticFeedback |

### 后端：Node.js Express
| 层 | 技术 |
|----|------|
| 框架 | Express |
| 数据库 | SQLite |
| 认证 | JWT + bcryptjs |
| 安全 | Helmet + express-rate-limit |
| AI | DeepSeek Flash API |
| 城市搜索 | 高德地图API |
| 天气数据 | Open-Meteo |
| 反向代理 | Nginx (端口8888→后端) |

## 四、数据存储分类

| 位置 | 数据 |
|------|------|
| 云端 SQLite | 用户账号、心情、日记、签到、友人、树洞、胶囊、古诗库、天气缓存、推送缓存 |
| 本地 sqflite | 照片路径、AI精灵成长数据 |
| 本地 shared_prefs | token、主题、通知开关、白噪音音量、定位缓存 |

## 五、内存与包体积目标

- APK ≤ 30MB（split per ABI）
- 运行内存 ≤ 150MB（空闲 ≤ 60MB）
- 策略：Canvas自绘替代图片、64kbps AAC音频、懒加载、tree-shaking、不用额外字体
