# 心晴日记 — UI设计规范

> 版本：v1.0 | 更新：2026-05-12

## 一、设计原则

- iOS/ColorOS 极简风格，干净克制有温度
- 不使用emoji作为功能图标 → Material Icons
- 暖金色为主调，不搞赛博朋克紫蓝粉
- "夜里的暖灯" ——不是"赛博网吧"

## 三、霓虹登录页

- 底色：`#0A0A0A`
- 霓虹线条：暖金 `#C4A46C`
- 入场动画：Logo逐笔画圆+射线(600ms ease-out)
- 输入框聚焦：边框亮度60%→100%(300ms) + 12px暖金辉光
- 按钮：表单完整时心跳脉冲(1200ms循环)
- 错误：淡红小字淡入，不弹窗打断

## 四、动画标准

| 类型 | 缓动 | 时长 | 场景 |
|------|------|------|------|
| 入场 | `ease-out (0.33,0,0.1,1)` | 300ms | 弹窗、页面、卡片 |
| 退出 | `ease-in (0.33,0,0.67,1)` | 250ms | 弹窗关闭 |
| 微交互 | `ease-out` | 150ms | 按钮、涟漪 |
| 溶解过渡 | `ease-out` | 600ms | 主题切换 |
| 轮播吸附 | `ease-out` | 400ms | 3D天气轮播 |

## 五、触感反馈

- 按钮：`lightImpact()`
- 保存成功：`mediumImpact()`
- 签到：`heavyImpact()`
- 错误：`lightImpact()` × 2

## 六、视觉规范

- 卡片圆角：16px
- 按钮圆角：12px
- 标签/胶囊圆角：16px
- 主按钮高度：48dp
- 页面padding：16dp
- 卡片padding：16dp
- 卡片间距：12dp
- 文字行高：1.5
- 触摸目标：≥48dp（Android）/ 44pt（iOS）
- 字体：系统默认Roboto，不引入额外字体
