# 心晴日记 — 产品需求文档 (PRD)

> 版本：v1.0 | 更新：2026-05-12

## 一、产品定位

Flutter 手机端天气+心情记录 App。iOS/ColorOS 风格，动画流畅与质量并重。手机端专属。

核心原则：**"你的情绪数据只属于你自己"**

## 二、信息架构（5 Tab）

| Tab | 名称 | 功能 |
|-----|------|------|
| 1 | 天气 | 仪表盘：3D弧形天气轮播、城市搜索、签到、天气心情卡、AI小精灵 |
| 2 | 心情 | 记录+回顾：8款情绪、碎碎念、照片、AI回应+古诗、日记、时间轴 |
| 3 | 树洞 | 匿名温暖：匿名留言、云拥抱/云咖啡、敏感词过滤 |
| 4 | 友人 | 轻社交：手机号加好友、查看好友心情（需授权7天过期） |
| 5 | 我的 | 个人中心：设置、时光胶囊、白噪音、壁纸、合规 |

## 三、核心功能清单

### 3.1 登录与注册
- 暖金单色霓虹UI（深黑底 #0A0A0A + 暖金 #C4A46C）
- 手机号作为账号
- 3种安全问题题型（选择/日期/自定义）
- 首次使用：起用户名 → 权限公告 → 功能引导气泡

### 3.2 天气与定位
- 4层定位：本地缓存 → GPS → IP定位 → 默认北京
- 高德地图API城市搜索（覆盖全国地级市+县级市）
- Open-Meteo 3天天气数据
- 弧形3D轮播卡片 + Canvas天气动画 + 音效

### 3.3 心情记录
- 8款基础情绪（单选）：开心/平静/难过/生气/焦虑/疲惫/期待/思念
- 情绪标签（多选，跟基础情绪动态切换）
- 碎碎念 + 照片上传（本地存储）
- DeepSeek Flash AI回应 + 古诗匹配
- 负面情绪三阶梯干预
- 日记（心情记录扩展）

### 3.4 签到与卡片
- 每日签到 + 连续签到计数
- 每日天气心情卡（自动生成、可保存/分享）
- 连续7天解锁限定壁纸（季节/节日/纪念日关联、有期限）

### 3.5 可视化
- 心情曲线图（fl_chart，周/月视图，叠加天气图标）
- 情绪时间轴（与曲线双向联动）
- 情绪分布饼图
- 记忆相册（季节合辑+那年今日+心情地图）

### 3.6 AI小精灵
- 暖金像素小狐狸形象
- 5阶段成长：光点→雏形→幼体→成长→亲密
- 根据情绪变化动作 + 点击气泡互动
- 本地AI数据存储（越用越懂你）

### 3.7 白噪音
- 6种：雨声/篝火/海浪/森林鸟鸣/风铃/心跳
- 播放/暂停/音量 + 后台播放 + 最小化悬浮球

### 3.8 轻社交
- 手机号搜好友 + 请求/同意流程
- 查看好友近3天心情（需授权、7天过期）
- 按好友情绪匹配提示文案
- 树洞匿名留言（登录后、每天3条上限、敏感词过滤）
- 云拥抱/云咖啡互动

### 3.9 时光胶囊
- 给3天或7天后的自己留一句话
- 到期本地通知 + 拆信封动画

### 3.10 推送通知
- 早安7:00（天气+AI早安语+古诗）
- 晚安22:00（AI回应当天心情）
- 时光胶囊到期通知

### 3.11 设置与合规
- 修改用户名/密码/手机号
- 3套主题（暖白基础/护眼柔绿/深夜暗金）
- 通知开关/权限管理
- 隐私政策+用户协议+ICP备案号
- 注销账号

## 四、数据存储原则

| 存储位置 | 数据 |
|----------|------|
| 云端（服务器SQLite） | 账号、心情记录、日记、签到、友人、树洞、胶囊、古诗库 |
| 本地（手机） | 照片、AI精灵数据、主题偏好、壁纸、离线缓存、token |

- 照片绝对不上传服务器
- API key存服务器环境变量，永不写进App代码
